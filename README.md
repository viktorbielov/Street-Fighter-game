
##Game instalation. Run:

`npm install`

`npm run build`

`npm run start`

Open http://localhost:9000/

##Playing controls:

left player: 
A - Hit <br />
D - Block <br />
Q+W+E - Critical hit(block doesn't affect it)

right player: 
J - Hit <br />
L - Block <br />
U+I+O - Critical hit(block doesn't affect it)
